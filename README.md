# Customer_Shopping_Behaviour_Analysis
Data Analytics Project analyzing customer shopping behavior using Python, SQL and Power BI

🔍 Overview

This project focuses on analyzing customer shopping behavior using end-to-end data analytics workflows. It covers data ingestion, exploratory data analysis (EDA), data cleaning, SQL-based querying, and dashboard creation to generate actionable business insights.

The goal is to understand purchasing patterns, customer segmentation, and key drivers of revenue to support data-driven decision-making.

📁 Dataset

Retail transaction dataset containing customer purchases, product categories, and transaction details

Includes features such as:

Customer ID

Product category

Purchase amount

Transaction date

Frequency of purchases

🛠️ Tools & Technologies

Python: Pandas, NumPy, Matplotlib, Seaborn

SQL: PostgreSQL / MySQL / SQL Server (Joins, Aggregations, CTEs)

Data Visualization: Power BI (DAX, Power Query)

Reporting: Power BI Report, Gamma (PPT generation)

Environment: Jupyter Notebook

⚙️ Project Workflow
1. Data Loading

Imported dataset using Python (Pandas)

Performed initial inspection (shape, columns, data types)

2. Exploratory Data Analysis (EDA)

Analyzed distributions of purchase amounts and categories

Identified trends in customer behavior

Visualized patterns using plots and charts

3. Data Cleaning

Handled missing values and duplicates

Standardized formats (dates, categories)

Removed outliers for better analysis

4. SQL Analysis

Loaded cleaned data into relational databases

Wrote SQL queries to extract insights:

Customer segmentation

Revenue trends

Product performance

Used joins, aggregations, and window functions

5. Dashboard Development

Built interactive Power BI dashboard

Key features:

Sales trends over time

Top-performing products

Customer segmentation insights

KPI tracking (revenue, frequency, growth)

6. Reporting & Presentation

Created a detailed analytical report summarizing insights

Designed a professional presentation using Gamma

Highlighted key findings and business recommendations

📊 Dashboard Highlights

Interactive filters for customer segments and product categories

Visual KPIs for revenue, growth, and purchase frequency

Drill-down capabilities for detailed analysis

Clean and intuitive UI for business stakeholders

📈 Key Results

Identified high-value customer segments driving revenue

Discovered trends in product demand and seasonal behavior

Improved understanding of purchase frequency and retention patterns

Enabled data-driven recommendations for marketing and sales strategies
